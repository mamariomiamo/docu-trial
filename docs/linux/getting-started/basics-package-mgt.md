## Check Installed Versions

`apt policy package_name`

## Check Content of the Package

Installed package: 
`dpkg -L package_name`

deb file: 
`dpkg -c package_file.deb`

without installing and without the deb file
```bash
sudo apt-file update        
apt-file list package_name
```
